# latkerlo-jvotci changelog

2.x added support for putting zi'evla in lujvo, fixed a bunch of bugs that introduced, and added a CLI.

1.x was the original version that only supported CLL lujvo.

For more details see <https://github.com/latkerlo/latkerlo-jvotci>. Also note that latkerlo-jvotci existed as four
separate versions, one for each programming language (Python, JavaScript, TypeScript, and Rust). In jvot3 we avoid
this by just writing Rust targeting WebAssembly.
